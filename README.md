## Runnig locally
1. Download repository with: `git clone https://github.com/JFRNogueira/2048-game.git`
2. Navigate to project folder: `cd 2048-game`
3. Install dependencies with `npm install`
4. Start project `npm start`





## Deploy application
1. Download repository with: `git clone https://github.com/JFRNogueira/2048-game.git`
2. Navigate to project folder: `cd 2048-game`
3. Install dependencies with `npm install`
4. Start project `npm start`
